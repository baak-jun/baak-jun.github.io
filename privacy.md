# 개인정보처리방침 (Privacy Policy)

**최종 수정일:** 2024년 05월 20일 (오늘 날짜로 수정하세요)

## 1. 개요
**ShortsBlock** (이하 "본 확장 프로그램")은 사용자의 개인정보를 최우선으로 보호합니다. 본 확장 프로그램은 사용자의 개인정보를 수집, 저장, 공유하지 않음을 명확히 밝힙니다.

## 2. 데이터 수집 및 전송 (Data Collection)
본 확장 프로그램은 사용자의 어떠한 식별 가능한 개인정보(이름, 주소, 전화번호, 이메일, 브라우징 기록, 쿠키 등)도 외부 서버로 전송하거나 수집하지 않습니다.

## 3. 데이터 저장 (Data Storage)
사용자가 설정한 옵션(일일 제한 시간, 차단 여부 등)은 사용자의 브라우저 내부에 있는 로컬 저장소(`Chrome Storage API`)에만 저장됩니다. 이 데이터는 사용자의 기기를 벗어나지 않으며, 확장 프로그램을 삭제할 경우 즉시 영구적으로 삭제됩니다.

## 4. 권한 사용 (Permissions)
본 확장 프로그램은 기능을 수행하기 위해 최소한의 권한만을 요청합니다.
*   **Storage:** 설정값을 로컬에 저장하기 위함.
*   **Tabs & WebNavigation:** 유튜브 페이지 이동을 감지하여 쇼츠를 차단하기 위함.
*   **Host Permissions (YouTube):** 유튜브 화면 내의 쇼츠 요소를 식별하고 제거하기 위함.
